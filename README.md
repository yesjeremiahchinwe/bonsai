# Bonsai Clone With Bootstrap and JavaScript

This project is a clone of the Bonsai official website having a total of 10 pages
- Home page
- Pricing
- small business page
- Agencies page
- Freelancers page
- Contracts page
- Collaboration page
- Payments
- Consulting
- Coaching
